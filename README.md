# ShootingProject_2_unity

## コンセプト
ゲーム性に重点を置き、短時間でプレイヤーの自己成長が実感できるシューティング。

1ステージは短時間で、ストーリーに関連した複数ステージを攻略する。

### ゲームプレイ
敵弾に掠るとスコアアップ&ボムが手に入るシステムを採用し、ゲーム性を向上させる。

収集要素として、ステージクリア報酬としてショットを得られるようにする。

操作キャラクターの能力的成長、プレイヤーの技術的成長を明確化。

## ゲーム概要
- 想定プレイ時間：1ステージ約5分
- ジャンル：2D縦スクロールシューティング
- プラットフォーム：PC(Windows)

## 操作方法
- 上下左右キー：移動
- Zキー：ショット
- Xキー：ボム
- LSHIFTキー：低速化

## 使用技術
- Unity C#
- Jsonデータ管理
